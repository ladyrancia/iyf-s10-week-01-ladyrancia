# Week 1: Web Foundations

Repository: iyf-s10-week-01-ladyrancia

## Overview
This project is part of Week 1 of the Web Foundations module.  
The goal was to learn the basics of web development, create a simple website using HTML, and deploy it using GitHub Pages.

By the end of the week I created a multi-page portfolio website that includes:
- Home page
- About page
- Projects page
- Contact page

The website demonstrates basic HTML structure, semantic HTML, accessibility improvements, and a working contact form.
